# 4.1 基本积分表

在以下积分公式中，$C$ 代表任意常数，$\alpha$ 为任意实数且 $\alpha \ne -1$， $a$ 和 $k$ 为非零常数。

## 一、基础函数积分公式

这些公式是微分学的逆运算，是积分计算的核心。

1.  $$\int 0 \, \mathrm{d} x = C$$
    
2.  $$\int k \, \mathrm{d} x = k x + C$$
    
3.  $$\int x^{\alpha} \, \mathrm{d} x = \frac{x^{\alpha+1}}{\alpha+1} + C \quad (\alpha \ne -1)$$
    
4.  $$\int \frac{1}{x} \, \mathrm{d} x = \ln |x| + C$$
    
5.  $$\int a^x \, \mathrm{d} x = \frac{a^x}{\ln a} + C \quad (a > 0, a \ne 1)$$
    
6.  $$\int e^x \, \mathrm{d} x = e^x + C$$
    

## 二、三角函数积分公式

7.  $$\int \cos x \, \mathrm{d} x = \sin x + C$$
    
8.  $$\int \sin x \, \mathrm{d} x = -\cos x + C$$
    
9.  $$\int \sec^2 x \, \mathrm{d} x = \tan x + C$$
    
10.  $$\int \csc^2 x \, \mathrm{d} x = -\cot x + C$$
     
11.  $$\int \sec x \tan x \, \mathrm{d} x = \sec x + C$$
     
12.  $$\int \csc x \cot x \, \mathrm{d} x = -\csc x + C$$
     

## 三、常用扩展积分公式

这些公式在涉及分式和三角函数时非常常用。

13.  $$\int \tan x \, \mathrm{d} x = -\ln |\cos x| + C$$
     
14.  $$\int \cot x \, \mathrm{d} x = \ln |\sin x| + C$$
     
15.  $$\int \sec x \, \mathrm{d} x = \ln |\sec x + \tan x| + C$$
     
16.  $$\int \csc x \, \mathrm{d} x = \ln |\csc x - \cot x| + C$$
     

## 四、反三角和复杂根式积分公式

这些积分结果通常用于有理分式和根式代换后的计算。

17.  $$\int \frac{1}{a^2 + x^2} \, \mathrm{d} x = \frac{1}{a} \arctan \left(\frac{x}{a}\right) + C$$
     
18.  $$\int \frac{1}{\sqrt{a^2 - x^2}} \, \mathrm{d} x = \arcsin \left(\frac{x}{a}\right) + C \quad (a > 0)$$
     
19.  $$\int \frac{1}{\sqrt{x^2 \pm a^2}} \, \mathrm{d} x = \ln |x + \sqrt{x^2 \pm a^2}| + C$$
     
20.  $$\int \frac{1}{x^2 - a^2} \, \mathrm{d} x = \frac{1}{2a} \ln \left| \frac{x - a}{x + a} \right| + C$$
     

## 五、特殊积分公式

这些公式通常是使用分部积分法（如第 21, 22 项）或三角代换（如第 23, 24 项）得到的，建议记住结果。

21.  $$\int \ln x \, \mathrm{d} x = x \ln x - x + C$$
     
22.  $$\int \arctan x \, \mathrm{d} x = x \arctan x - \frac{1}{2} \ln (1+x^2) + C$$
     
23.  $$\int \sqrt{a^2 - x^2} \, \mathrm{d} x = \frac{x}{2} \sqrt{a^2 - x^2} + \frac{a^2}{2} \arcsin \left(\frac{x}{a}\right) + C$$
     
24.  $$\int \sqrt{x^2 \pm a^2} \, \mathrm{d} x = \frac{x}{2} \sqrt{x^2 \pm a^2} \pm \frac{a^2}{2} \ln |x + \sqrt{x^2 \pm a^2}| + C$$
